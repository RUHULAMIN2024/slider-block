A powerful, customizable, and responsive image and content slider built natively for the WordPress Block Editor (Gutenberg).

== Description ==

**Ruhulamin Slider Block** is a blazing-fast, customizable, and highly interactive content and image slider designed specifically for the modern WordPress Block Editor (Gutenberg).

Powered by the robust Swiper JS engine, it provides endless possibilities to create stunning hero sections, product showcases, testimonial sliders, or simple image galleries directly within the editor--with a live visual preview.

### 🌟 Key Features
* **Fully Visual Drag & Drop Editor:** Position your slide title, description, and buttons effortlessly using intuitive drag-and-drop right inside the Gutenberg editor!
* **Multiple Stunning Effects:** Choose from standard Slide, Fade, Cube, Creative, Coverflow, Flip, and Cards effects to make your sliders pop!
* **Advanced Typography & Coloring:** Choose custom colors, backgrounds, and typography for every single element natively.
* **Fully Responsive:** Extensive device-specific controls. Set custom column gaps, slide views, and toggle pagination/navigation visibility separately for Desktop, Tablet, and Mobile devices.
* **Autoplay & Loop Controls:** Smooth autoplay with intelligent interactions (pause on hover, disable on interaction), infinity loop, and free-mode scrolling.
* **Accessibility Focused:** Keyboard and mouse-wheel control support out-of-the-box.

### 🚀 How to Use?
1. Edit any page or post.
2. Click the `+` icon to add a new block.
3. Search for **Ruhulamin Slider Block** and insert it.
4. Add your slides, customize the text, style the buttons, and position them by dragging!

== Installation ==

1. Upload the `ruhulamin-slider-block` folder to your `/wp-content/plugins/` directory, or install directly through the WordPress plugins screen.
2. Activate the plugin through the **Plugins** screen in WordPress.
3. Go to any post or page and add the **Ruhulamin Slider Block** block!

== Development and Source Code ==

The full source code and build tools for this plugin are publicly available on GitHub:
https://github.com/RUHULAMIN2024/slider-block

To regenerate the compiled assets from the source code, please follow these steps:
1. Ensure you have Node.js and npm installed on your system.
2. Navigate to the plugin directory via terminal.
3. Run `npm install` to install all dependencies.
4. Run `npm run build` to compile the block assets.

== Frequently Asked Questions ==

= Does it support responsive layouts? =
Yes! You can configure the number of columns, column gaps, and even toggle navigation arrows and pagination specifically for Desktop, Tablet, and Mobile.

= Can I use it for a full-width hero slider? =
Absolutely. The block supports WordPress's native Wide and Full-width alignments, making it perfect for immersive edge-to-edge hero sections.

= Is it fast? =
Yes, Ruhulamin Slider Block is optimized to load assets only when the block is present on the page, ensuring zero bloat on your site's performance.
